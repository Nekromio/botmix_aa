# [BotMiX] Admin Action Info

SourceMod-плагин для серверов Source-игр, который отправляет информацию
о действиях администраторов (бан, мут, гаг и т.п.) в связанные чаты
через ядро **BotMiX**.

## Возможности
- Поддержка BaseComm / MaterialAdmin / SourceBans++
- Автоопределение используемой админ-системы
- Отправка уведомлений в несколько чатов одновременно
- Маскирование чувствительных данных (опционально)
- Выбор канала отправки сообщений
- Логирование в файл

## Требования
- SourceMod 1.11+
- Ядро **BotMiX** (закрытое)
- Одна из админ-систем:
  - BaseComm
  - MaterialAdmin
  - SourceBans++

## Установка
1. Установить ядро **BotMiX**  
   Скачать: https://hlmod.net/threads/botmix-vk-tg-server-otkrytoe-testirovanie.67545/
2. Скомпилировать или установить `botmix_aa.smx`
3. Перезапустить сервер или сменить карту

## Конвары
```cfg
sm_botmix_aa_show_info 1   // Показывать данные полностью (0 — маска ***123)
sm_botmix_aa_channel   2   // Канал отправки сообщений (0–9)
